Get started

The Angola web Api provides programmatic access to data about Angola country. Retrieve radio stations , wheather , restaurants and tourist atractions (Last 2 to be developed soon ) and filter them .

To use this API, you don't need an API key. Please contact us at kawakuticode@gmail.com if you want help the project.

Radio Stations

To get list of Radio stations from Angola you need to make a GET call to the following url :
https://angolawebapi.herokuapp.com


QUERY PARAMETERS

Field Type Description
/v1/radios Endpoint get a list of radios from angola
/v1/radios/radio_name String search radio by name

Weather

To get weather of the provinces of Angola you need to make a GET call to the following url :
https://angolawebapi.herokuapp.com


QUERY PARAMETERS

Field Type Description
/v1/weather String get a list with weather of all provinces from angola
/v1/weather/province name String search weather by province name

Errors

The Angolan API uses the following error codes:

Error Code Meaning
404 parameter invalid. This error appears when you pass parameter not valid for the query.
